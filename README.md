# wombat-oam-cookbook

A Chef cookbook for installing and configuring WombatOAM

## Supported Platforms

* Ubuntu 12.04

## Usage

### wombat-oam::default

Include `wombat-oam` in your node's `run_list`:

```json
{
  "run_list": [
    "recipe[wombat-oam::default]"
  ]
}
```
